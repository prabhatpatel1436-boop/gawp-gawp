# BonelzBot Premium Selling Website - Design Guidelines

## Design Approach

**Reference-Based Approach:** Drawing inspiration from redline.tools, gaming automation platforms, and premium SaaS landing pages. This combines the sleek, modern aesthetic of gaming products with the conversion-focused structure of top-tier selling websites.

**Key Design Principles:**
- Premium gaming aesthetic with depth and dimension
- Trust-building through visual proof (screenshots, stats, testimonials)
- Clear conversion paths leading to Discord purchase
- Professional polish that justifies premium pricing

## Core Design Elements

### A. Typography System

**Font Families:**
- Primary: Inter (via Google Fonts) - for body text, UI elements, and secondary headings
- Display: Space Grotesk or Orbitron (via Google Fonts) - for hero headlines and major section titles, creating that modern tech/gaming feel

**Typography Hierarchy:**
- Hero Headline: 4xl/5xl/6xl font size, bold weight, tight letter spacing
- Section Titles: 3xl/4xl, semibold, standard tracking
- Feature Headings: xl/2xl, medium weight
- Body Text: base/lg, regular weight, relaxed line height (1.7)
- Captions/Labels: sm, medium weight, uppercase with wide tracking for emphasis
- Button Text: base, semibold

### B. Layout System

**Spacing Primitives:**
Use Tailwind units of **4, 6, 8, 12, 16, 20, 24** for consistency:
- Component spacing: p-4, p-6, p-8
- Section padding: py-16, py-20, py-24
- Element gaps: gap-4, gap-6, gap-8
- Margin separation: mb-8, mb-12, mb-16

**Grid System:**
- Container: max-w-7xl with px-6 for comfortable reading
- Feature grids: grid-cols-1 md:grid-cols-2 lg:grid-cols-4 (for 4 primary features)
- Pricing cards: grid-cols-1 md:grid-cols-2 (two pricing options side-by-side)
- FAQ: Single column max-w-3xl for optimal readability
- Statistics: grid-cols-2 md:grid-cols-3 for key metrics

**Vertical Rhythm:**
- Hero section: 90vh minimum height for impact
- Content sections: py-20 (desktop), py-12 (mobile)
- Inter-section spacing: mb-0 (sections flow continuously)

### E. Animations

**Strategic Animation Use (minimal, purposeful):**
- Hero orbs: Slow floating drift (20-30 second loop)
- Feature cards: Subtle hover lift (translateY -2px)
- Pricing cards: Gentle hover shadow expansion
- FAQ accordions: Smooth height transition (300ms)
- Scroll reveal: Fade-in on sections as they enter viewport (optional enhancement)
- CTA buttons: Standard hover scale (1.02x)

**NO animations for:**
- Background elements (beyond hero orbs)
- Text content
- Images/screenshots
- Section transitions

## Multi-Column Strategy

**Desktop Layout (lg:):**
- Features: 4 columns
- Statistics: 3 columns
- Pricing: 2 columns
- FAQ: 1 column (max-w-3xl)
- Footer: 3 columns

**Tablet (md:):**
- Features: 2 columns
- Statistics: 3 columns
- Pricing: 2 columns
- FAQ: 1 column
- Footer: 2 columns

**Mobile (base):**
- All sections: 1 column stack
- Maintain generous spacing (py-12)

## Accessibility & Polish

- All interactive elements have focus states
- Form inputs (if any) maintain consistent styling
- Sufficient contrast ratios throughout
- Clear visual hierarchy with typography scale
- Touch targets minimum 44px for mobile
- Semantic HTML structure (proper heading levels)
- Discord link opens in new tab with appropriate attributes

- More comprehensive feature showcase (8 features vs 4)

- Feature deep-dive with alternating layouts
- Enhanced pricing presentation with currency toggle
- More detailed FAQ section (10 Q&As)
- Stronger Discord community integration
- Professional screenshot presentation in frames
- Statistics banner for immediate credibility
- Multiple conversion points throughout journey

This creates a premium, conversion-optimized selling website that positions BonelzBot as the professional choice for Clash of Clans automation.
